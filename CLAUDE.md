# 프로젝트: 글쓰기 성장 플랫폼 (writing-growth)

매일 글쓰기 주제를 제시하고 AI 피드백을 주는 웹앱. 공익적 목적(글쓰기 진입장벽 낮추기)으로 만드는 개인 프로젝트, MVP 단계.

## 개발자 참고사항
- 개발자는 CS 1학년, 기초 문법은 알지만 프론트엔드(TS/HTML/CSS)는 아직 깊게 모름.
- 코드를 짤 때 화려한 트릭이나 과도한 추상화보다 **읽기 쉽고 단순한 코드**를 우선한다.
- 새 개념(예: hook, 상태관리 등)을 처음 도입할 때는 코드 위에 한 줄 주석으로 왜 쓰는지 간단히 남긴다.

## 기술 스택
- Next.js (App Router) + TypeScript + Tailwind CSS
- 배포: Vercel
- DB/인증 (예정): Supabase
- AI 피드백: Claude API

## 명령어
- 개발 서버: `npm run dev`
- 빌드: `npm run build`
- 린트: `npm run lint`

## 코드 컨벤션
- 스타일은 Tailwind 유틸리티 클래스만 사용, 별도 CSS 파일 만들지 않는다.
- 컴포넌트는 `app/components/`에, 페이지는 `app/`에 둔다.
- 스타일/포맷팅은 린터에 맡기고, 별도 스타일 규칙을 지시하지 않는다.

## 지금 범위 (MVP) — 완전 무료로 운영 목표
1. 오늘의 글쓰기 주제 표시 ✅
2. 텍스트 입력 + Supabase 저장 ✅
3. 지난 글 목록 보기 ← 지금 여기
4. 배포 (Vercel 무료 티어)

AI 피드백 기능은 이번 MVP에서 제외한다 (비용 발생 + 범위 축소 목적). Claude API 관련 코드를 먼저 제안하지 않는다.
로그인, 커뮤니티, 결제 기능도 이후 단계 — 지금은 다루지 않는다.

## 하지 말 것
- .env 파일에 있는 API 키를 코드에 하드코딩하거나 콘솔에 출력하지 않는다.
- 요청하지 않은 기능(로그인, 다국어 등)을 먼저 추가하지 않는다.
